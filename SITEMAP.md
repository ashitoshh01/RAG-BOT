# 🗺️ Project Sitemap — RAG Chatbot

A full map of every file and directory in the project, with a description of its role.

---

## 📁 Root Directory

```
RAG/
│
├── 🐍 Backend
│   ├── main.py              →  FastAPI app entry point; defines API routes & serves static files
│   ├── rag_pipeline.py      →  Core RAG logic: embeddings (Google), retrieval (ChromaDB), LLM (Groq)
│   ├── ingest_data.py       →  CLI script to ingest PDF/TXT documents into the ChromaDB vector store
│   └── ingest.py            →  Alternative/older ingestion script
│
├── 🌐 Frontend
│   ├── index.html           →  Chat UI (served by FastAPI at GET /)
│   ├── script.js            →  Handles user input, POST /chat API calls, and DOM updates
│   └── style.css            →  Chatbot UI styling
│
├── 🗄️ Data & Storage
│   ├── vectorstore/         →  ChromaDB persisted vector embeddings (auto-generated by ingest)
│   └── documents/           →  Place your source PDF/TXT documents here before ingesting
│
├── ⚙️ Config & Environment
│   ├── .env                 →  API keys (GROQ_API_KEY, GOOGLE_API_KEY) — never commit this
│   ├── .env.bak             →  Backup of previous .env
│   ├── .gitignore           →  Files excluded from git (venv, .env, __pycache__, etc.)
│   └── requirements.txt     →  Python package dependencies
│
├── 🖼️ Assets
│   └── favicon.png          →  Browser tab icon
│
└── 📦 Python Environment
    └── venv/                →  Virtual environment (not committed to git)
```

---

## 🔌 API Endpoints

| Method | Path         | Description                            |
|--------|--------------|----------------------------------------|
| GET    | `/`          | Serves `index.html` (chat UI)          |
| GET    | `/script.js` | Serves the frontend JavaScript         |
| GET    | `/style.css` | Serves the frontend CSS                |
| GET    | `/favicon.ico` | Serves the favicon (PNG format)      |
| GET    | `/favicon.png` | Serves the favicon PNG directly      |
| POST   | `/chat`      | Accepts `{ "message": "..." }`, returns `{ "answer": "..." }` |

---

## 🤖 AI Stack

| Component     | Provider / Model                          |
|---------------|-------------------------------------------|
| **LLM**       | Groq → `llama-3.3-70b-versatile`          |
| **Embeddings**| Google Generative AI → `gemini-embedding-001` |
| **Vector DB** | ChromaDB (local, persisted to `vectorstore/`) |
| **Framework** | LangChain (RAG chain, prompts, retrieval) |

---

## 🔄 Data Flow

```
User types message
       ↓
  index.html (frontend)
       ↓  POST /chat
  main.py (FastAPI)
       ↓
  rag_pipeline.py
       ↓                          ↓
  ChromaDB retriever      Groq LLM (llama-3.3-70b)
  (top-3 relevant chunks)
       ↓                          ↓
         → Prompt Template ←
               ↓
           Final Answer
               ↓
      Response JSON { "answer": "..." }
               ↓
         Chat UI renders reply
```
